# Engineering Logs

An engineering notebook documenting projects, experiments, failures, and lessons learned across robotics, software, and computer science.

## About

Engineering Logs is my personal platform where I document what I build, how I build it, and what I learn along the way. Instead of showcasing only finished projects, this website also records the engineering process—including successes, failures, debugging sessions, and design decisions.
